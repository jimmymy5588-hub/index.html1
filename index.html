Sinkron Cloud (Firebase)

1. Buat project di console.firebase.google.com (gratis)
2. Daftarkan Web App (ikon </>), salin objek firebaseConfig
3. Aktifkan Firestore Database mode Test mode
4. Tempel config di bawah, buat kode sinkron bebas — pakai kode yang sama di tiap perangkat
